# Action Items Tracker

Single-file HTML action tracker with List and Kanban views, filters, stats, and meeting-notes import.

## Deploy to Render

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/subsumeet-collab/action-tracker)

Or manually: in the Render dashboard, **New > Static Site**, connect this repo, leave the build command empty, and set the publish directory to `.`.
